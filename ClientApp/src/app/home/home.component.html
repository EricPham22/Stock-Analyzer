How it Works

Stock analyzer allows you to input a ticker symbol and determine wether is it a good buy, sell or hold according to different technical indicators. You can input up to 3 indicators. See Description for information on each indicator.
Ticker Symbol - The stock that you want to look up
Interval - The time interval between two consecutive data points
Indicator - The desired technical indicator
Time Period - The number of data points used to calcuate average
NOTE:
  • Using a free key from alpha vantage, Stock Analyzer only supports 1 search per minute
  • VWAP only supports IntraDay (1,5,15,30,60) interval value
